ANIMACE:

@keyframes blink {
    0% {
        něco něco něco;
    }

    50% {
        něco něco něco;
    }

    100% {
        něco něco něco;
    }
}

pro kompatibilitu na různých prohlížečích přidat toto
&-webkitkeyframes blink {
    0% {
        něco něco něco;
    }

    50% {
        něco něco něco;
    }

    100% {
        něco něco něco;
    }
}

něco, co bude ovlivněno něčím, a obsahuje:
animation: blink (nebo jakýkoliv jiný shodující se název) 10s infinite;
-webkit-animation: blink 10s infinite;
(https://www.youtube.com/watch?v=wfaDzSL6ll0&ab_channel=MDB-justcodeit)

gradienty:

lze ozkoušet na "cssgradient.io" a zkopírovat properties gradientu do css

nějaký objekt {
    něco
    background: barva;
    background: radial(nebo linear)-gradient(
        circle, (nebo něco)
        barva 0%,
        barva 100%
        );
}
